## Key Concepts Learned

### Machine Learning

- Machine Learning is a branch of Artificial Intelligence (AI).
- It enables computers to learn patterns from data without being explicitly programmed.
- ML models improve their performance by learning from data.

### Supervised Learning

- Learns from labeled data.
- Used for prediction tasks.
- Common applications include email spam detection, house price prediction, and disease diagnosis.

### Unsupervised Learning

- Learns from unlabeled data.
- Finds hidden patterns and relationships.
- Common applications include customer segmentation and anomaly detection.

### Reinforcement Learning

- Learns through trial and error.
- Uses rewards and penalties to improve decision-making.
- Common applications include robotics, game AI, and self-driving cars.

### Regression

- Predicts continuous numerical values.
- Examples include predicting house prices, temperatures, and sales.

### Classification

- Predicts categories or class labels.
- Examples include spam detection, disease diagnosis, and image classification.

### Decision Trees

- A supervised learning algorithm used for both classification and regression.
- Makes decisions using a tree-like structure consisting of root nodes, internal nodes, branches, and leaf nodes.
- Easy to understand and visualize.
