# Day 08 - Python Lists & List Operations

## Overview

This project is part of my Python learning journey. It focuses on understanding Python lists and applying them through practical projects.

---

## Projects

### 1. Student Grade Manager

**Features:**

- Add a student
- Remove a student
- Update grades
- Calculate average grade
- Find highest and lowest grades
- Display top 3 performers
- Filter students above and below average

### 2. Shopping Cart System

**Features:**

- Add items
- Remove items
- Update quantity
- Calculate total bill
- Apply discount
- Display receipt
- Show recently added items

### 3. Data Cleaning Pipeline

**Features:**

- Remove `None` values
- Remove duplicates
- Remove extra spaces
- Normalize text case
- Display data quality metrics
