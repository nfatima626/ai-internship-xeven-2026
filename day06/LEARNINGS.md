# Key Learnings

- Learned that lists are ordered and mutable collections used to store multiple values in a single variable.
- Understood how indexing and negative indexing work in Python.
- Learned how to access, update, and slice list elements efficiently.
- Understood the difference between `append()`, `insert()`, and `extend()`.
- Learned different techniques for removing elements using `remove()`, `pop()`, and `clear()`.
- Understood the difference between `sort()` and `sorted()`.
- Learned how to use membership operators (`in` and `not in`) to check whether an item exists in a list.
- Gained a clear understanding of nested lists and how to access rows and columns using multiple indexes.
- Learned the important difference between Copy and Reference and how shared references can unintentionally modify data.
- Learned how List Comprehension provides a concise and Pythonic way to create new lists.
